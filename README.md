# Forestation Planning & Suitability System( GVAN )

A data‑driven tool that helps evaluate forestation projects using soil quality, rainfall, climate, water availability, budget, and plantation goals.

## Installation
1. Install Python 3.9+
2. Run `pip install -r requirements.txt`
3. Run `streamlit run app.py`

## Inputs required
Land size, soil type, climate, rainfall, water source, budget, goal, maintenance level, optional preferred species.

## Output
Suitability index, recommended species, density, cost breakdown, success probability, carbon estimate, and risk analysis.
